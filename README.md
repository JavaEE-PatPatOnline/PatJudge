# PatJudge

> Judge backend for Patpat Online, made with 💖.

---

## 部署 & 运行

参见 [PatBoot](https://github.com/JavaEE-PatPatOnline/PatBoot)。PatJudge 需要访问题目配置目录和用户提交目录，因此需要首先在 `bucket` 中创建 `problem` 和 `submission` 目录。
